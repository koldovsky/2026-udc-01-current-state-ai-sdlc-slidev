# Cost and tokens

<div class="mt-2 text-[0.92em]">

- A **token** ≈ ¾ of a word; you pay for input + output, **pay-as-you-go**
- Reasoning and large context **multiply** the cost; don't dump the whole repo

| Code | 100 lines | 200 lines |
|---|---|---|
| React JSX | ~700 | ~1,500 |
| Python | ~1,000 | ~1,700 |
| SQL | ~1,150 | ~2,500 |

**Optimize:** curate context · cheap model for routine · `.cursorignore` / Repomix · **measure** tokens per feature

</div>
