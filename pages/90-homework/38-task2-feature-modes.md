# Task 2: Фіча через режими _(~30 хв)_

**Що зробити:**
- Обрати **одну невелику фічу** (напр. список нотаток / лічильник / проста форма)
- Сформулювати **acceptance criteria** (3-5 пунктів) перед кодом
- Пройти задачу у двох режимах: спершу **Plan**, потім **Agent** за погодженим планом
- Зробити рев'ю дифу; запустити й перевірити в браузері
- Описати флоу у `docs/workflow.md`: що робив у кожному режимі, що скоригував

**Verification:** фіча працює; `docs/workflow.md` описує plan → agent + рев'ю.

# Task 2: A feature via modes _(~30 min)_

**What to do:**
- Pick **one small feature** (e.g. a notes list / counter / simple form)
- Write **acceptance criteria** (3-5 points) before any code
- Run the task in two modes: **Plan** first, then **Agent** on the agreed plan
- Review the diff; run and check in the browser
- Document the flow in `docs/workflow.md`: what you did in each mode, what you adjusted

**Verification:** the feature works; `docs/workflow.md` describes plan → agent + review.
