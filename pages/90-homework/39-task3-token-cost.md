# Task 3: Аналіз токенів і вартості _(~25 хв)_

**Що зробити:**
- Зафіксувати споживання токенів за фічу (usage у IDE) і/або `npx repomix` для розміру контексту
- Прикинути **вартість** (токени × ціна обраної моделі)
- Знайти **2-3 оптимізації**: вузький контекст, дешевша модель для рутини, коротші ітерації
- (Опц.) повторити фічу з оптимізацією та порівняти
- Записати у `docs/cost-analysis.md`: цифри + висновки

**Verification:** `docs/cost-analysis.md` містить реальні числа й мінімум 2 висновки.

# Task 3: Token & cost analysis _(~25 min)_

**What to do:**
- Capture token usage for the feature (IDE usage view) and/or `npx repomix` for context size
- Estimate the **cost** (tokens × the chosen model's price)
- Find **2-3 optimizations**: narrower context, cheaper model for routine, shorter iterations
- (Opt.) redo the feature with the optimization and compare
- Write it up in `docs/cost-analysis.md`: numbers + findings

**Verification:** `docs/cost-analysis.md` has real numbers and at least 2 findings.
