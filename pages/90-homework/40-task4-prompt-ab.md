# Task 4 (бонус): A/B промптів _(~15 хв)_

**Що зробити:**
- Взяти одну й ту саму задачу і сформулювати **два промпти**:
  - базовий («зроби X»)
  - структурований (роль + контекст + обмеження + acceptance criteria + формат)
- Виконати обидва, зафіксувати різницю: якість, кількість ітерацій, токени
- Записати у `docs/ab-experiment.md`: обидва промпти + короткий висновок

**Verification:** `docs/ab-experiment.md` містить обидва промпти й порівняння.

<div class="mt-3 text-sm opacity-75">Бонус — не обов'язковий, але дає +розуміння для Воркшопу 2 (промпт-інженерія).</div>

# Task 4 (bonus): Prompt A/B _(~15 min)_

**What to do:**
- Take the same task and write **two prompts**:
  - basic ("do X")
  - structured (role + context + constraints + acceptance criteria + format)
- Run both, capture the difference: quality, number of iterations, tokens
- Write it up in `docs/ab-experiment.md`: both prompts + a short conclusion

**Verification:** `docs/ab-experiment.md` contains both prompts and a comparison.

<div class="mt-3 text-sm opacity-75">Bonus — optional, but a head start for Workshop 2 (prompt engineering).</div>
