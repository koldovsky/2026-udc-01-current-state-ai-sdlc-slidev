# Режими роботи Agentic IDE

<div class="grid grid-cols-2 gap-x-8 gap-y-2 mt-4 text-[0.95em]">

<div><strong>Chat / Ask</strong> — питання, пояснення коду, без змін у файлах</div>
<div><strong>Plan</strong> — спершу план/спека, потім виконання; менше «сюрпризів»</div>
<div><strong>Agent</strong> — багатокрокове виконання: редагує файли, запускає команди</div>
<div><strong>Debug</strong> — гіпотези + перевірка на рантаймі, аналіз помилок</div>

</div>

</div>

<div class="lang-en">

# Agentic IDE working modes

<div class="grid grid-cols-2 gap-x-8 gap-y-2 mt-4 text-[0.95em]">

<div><strong>Chat / Ask</strong> — questions, code explanation, no file changes</div>
<div><strong>Plan</strong> — plan/spec first, then execute; fewer "surprises"</div>
<div><strong>Agent</strong> — multi-step execution: edits files, runs commands</div>
<div><strong>Debug</strong> — hypotheses + runtime checks, error analysis</div>

</div>

</div>

<v-clicks>

- <span class="lang-uk">Правило: **складніша/ризикованіша задача → починай з Plan**, не одразу Agent</span><span class="lang-en">Rule: **harder/riskier task → start with Plan**, not Agent right away</span>
- <span class="lang-uk">Прості локальні правки — Agent напряму; питання — Chat</span><span class="lang-en">Small local edits — Agent directly; questions — Chat</span>
- <span class="lang-uk">Режими є в Cursor, Claude Code, GitHub Copilot (назви відрізняються)</span><span class="lang-en">These modes exist in Cursor, Claude Code, GitHub Copilot (names differ)</span>
