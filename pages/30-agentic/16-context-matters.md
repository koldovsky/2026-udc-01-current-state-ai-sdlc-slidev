<div class="lang-uk">

# Контекст вирішує все

</div>

<div class="lang-en">

# Context is everything

</div>

<v-clicks>

- <span class="lang-uk">LLM не «знає» ваш проєкт — вона знає лише те, що **в контексті прямо зараз**</span><span class="lang-en">An LLM doesn't "know" your project — only what's **in context right now**</span>
- <span class="lang-uk">Якість відповіді ≈ якість контексту: правильні файли, чіткі вимоги, приклади</span><span class="lang-en">Answer quality ≈ context quality: the right files, clear requirements, examples</span>
- <span class="lang-uk">Замало контексту → здогадки й галюцинації; забагато → шум, втрата фокусу, дорожче</span><span class="lang-en">Too little context → guessing and hallucinations; too much → noise, lost focus, higher cost</span>
- <span class="lang-uk">**Curate, не dump:** релевантні файли, правила (rules), приклади, acceptance criteria</span><span class="lang-en">**Curate, don't dump:** relevant files, rules, examples, acceptance criteria</span>
- <span class="lang-uk">Це фундамент усіх наступних тем: rules, commands, skills, MCP, SDD</span><span class="lang-en">This is the foundation for every later topic: rules, commands, skills, MCP, SDD</span>

</v-clicks>
