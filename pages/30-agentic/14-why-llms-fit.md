<div class="lang-en">

# Why LLMs are a good fit for software dev

</div>

<v-clicks>

- <span class="lang-uk">**Computer Science добре вивчена** — побудована людьми на основі математики</span><span class="lang-en">**Computer Science is well studied** — built by humans on top of math</span>
- <span class="lang-uk">Комерційна розробка — це **інженерія**, а не наука: ми **застосовуємо** відоме, а не винаходимо нове</span><span class="lang-en">Commercial dev is **engineering**, not science: we **apply** known knowledge, not invent new</span>
- <span class="lang-uk">Мови програмування по суті **прості** у фундаменті (напр. Go — 25 ключових слів)</span><span class="lang-en">Programming languages are fundamentally **simple** (e.g. Go has 25 keywords)</span>
- <span class="lang-uk">Типові патерни, архітектури, best practices **відомі** — важливо правильно застосувати</span><span class="lang-en">Common patterns, architectures, best practices are **well-known** — apply them properly</span>
- <span class="lang-uk">**Детермінованість** ПЗ спрощує тестування й перевірку коректності</span><span class="lang-en">**Determinism** of software simplifies testing and proving correctness</span>
- <span class="lang-uk">Величезний корпус коду на GitHub + Q&A на StackOverflow для тренування</span><span class="lang-en">A huge corpus of code on GitHub + Q&A on StackOverflow to train on</span>

</v-clicks>
