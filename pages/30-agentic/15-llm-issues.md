<div class="lang-uk">

# Де LLM ламаються в розробці

</div>

<div class="lang-en">

# Where LLMs break in development

</div>

<v-clicks>

- <span class="lang-uk">**Нерівномірність:** популярні стеки — добре, «домашні»/нішеві фреймворки — погано</span><span class="lang-en">**Uneven skill:** great on popular stacks, poor on "domestic"/niche frameworks</span>
- <span class="lang-uk">**Лаг тренувальних даних:** проблеми з новими версіями мов, фреймворків, бібліотек</span><span class="lang-en">**Training-data lag:** trouble with new versions of languages, frameworks, libs</span>
- <span class="lang-uk">**Галюцинації (hallucinations):** вигадують неіснуючі API, методи, пакети</span><span class="lang-en">**Hallucinations:** invent non-existent APIs, methods, packages</span>
- <span class="lang-uk">**Контекст:** ліміт розміру + втрата деталей у великому контексті</span><span class="lang-en">**Context:** size limit + losing details in large context</span>
- <span class="lang-uk">**Недетермінованість:** той самий запит → різні відповіді</span><span class="lang-en">**Non-determinism:** same prompt → different answers</span>
- <span class="lang-uk">**Безпека й legal:** вразливий/копірайтнутий код, витік даних, ширший attack-perimeter через інтеграції</span><span class="lang-en">**Security & legal:** vulnerable/copyrighted code, data leakage, wider attack surface via integrations</span>
- <span class="lang-uk">**Вартість:** інтенсивне використання буває дорогим</span><span class="lang-en">**Cost:** heavy use can get expensive</span>

</v-clicks>
